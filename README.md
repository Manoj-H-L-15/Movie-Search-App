🎬 Movie Search App

A modern and responsive Movie Search Application built using React.js that allows users to search and explore movies in real time. The application provides a clean and intuitive UI where users can instantly find movies by entering keywords.

🚀 Features
🔍 Real-time Movie Search
Search for movies by title and get instant results.
🎞️ Dynamic Movie Cards
Displays movie posters, titles, and release years in a clean card layout.
⚡ API Integration
Fetches live movie data from an external movie API.
🧩 Reusable Components
Built using modular and reusable React components.
📱 Responsive Design
Optimized for different screen sizes (desktop & mobile).

🛠️ Tech Stack
Frontend: React.js
Styling: CSS / Bootstrap
API: Movie Database API (TMDB or similar)
Version Control: Git & GitHub
Deployment: Netlify
💡 How It Works
Users enter a movie name in the search bar.
The app sends a request to the movie API.
Results are fetched and displayed dynamically as movie cards.
Each card shows:
Movie Poster
Movie Title
Release Year
